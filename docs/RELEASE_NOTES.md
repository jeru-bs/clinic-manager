# גרסה יציבה 1.0.0 — 2026-07-13

גרסה זו מוגדרת מוכנה לשימוש שוטף בדפדפן עבור ניהול מטופלים, מפגשים, יומן, משימות ותזכורות, תשלומים, קבצים, גיבוי ושחזור.

## נוסף בשחרור

- Audit אחיד לפעולות עסקיות ושמירת המשתמש שביצע את הפעולה.
- Undo אטומי לכל שינויי הגיליונות של פעולה אחת, כאשר הביטול בטוח.
- תזכורת בסיסית כחלק ממשימה ותצוגת תזכורות פעילות בדשבורד.
- גיליון `audit_log` שנוצר ונבדק דרך מנגנון תקינות הנתונים הקיים.
- אימות מבנה גיבוי ומזהים כפולים לפני כתיבה כלשהי.
- גיבוי חובה לפני שחזור ו-rollback אוטומטי אם השחזור נעצר באמצע.
- בדיקות תהליך לסיווג תקלות Google, Audit/Undo, תזכורות וגיבוי/שחזור.
- הגנה מפני דריסה שקטה בעריכה מקבילה: לפני עדכון או מחיקה נבדקת גרסת הרשומה העדכנית, ובמקרה התנגשות הנתונים מתרעננים והפעולה נעצרת.
- בדיקות דפדפן אוטומטיות ב-Chromium לתצוגת מחשב ומובייל, כולל שער הכניסה, תקלת חלון Google ומסך ההגדרות.
- תהליך CI מריץ את בדיקת השחרור המלאה בכל push ל-main ובכל Pull Request.

## גבולות ידועים

- מחיקת קבצים או פעולות שכבר שינו Google Drive/Calendar מתועדות ב-Audit, אך אינן מציעות Undo חלקי שאינו יכול להחזיר את המצב החיצוני במלואו.
- תמלול AI וחשבונית ירוקה אינם כלולים בשחרור, בהתאם להחלטת המוצר.
- חגים ישראליים מוזנים כחריגי יומן ידניים.

## תנאי שחרור

השחרור תקין כאשר `npm.cmd run release:check` עובר במלואו, בדיקת דפדפן מקומית עוברת, אין סודות בקבצים המנוהלים, והענף `main` נקי ומסונכרן לאחר הפרסום.
